# 调试模式
ArkPets 的调试热键一览。

### 前提
必须设置日志等级为 `DEBUG` 或在启动时指定 `--debug` 参数，以启用调试功能。

### 用法
| 热键           | 功能                                               |
|--------------|--------------------------------------------------|
| <kbd>D</kbd> | 显示当前帧率和 JVM 堆内存大小。                               |
| <kbd>P</kbd> | 显示平面调试信息。                                        |
| <kbd>S</kbd> | 进行截图操作，截图将保存到工作目录下的 `temp/snapshot-<当前时间戳>.png`。 |
| <kbd>W</kbd> | 显示系统窗口列表。                                        |
